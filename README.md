# kotlin-cli-maven-spring-failsafe-jacoco-cucumber-junit-car-data

## Description
A POC for spring app using junit5, cucumber,
and jacoco framework with failsafe and surefire plugins.

## Tech stack
- kotlin
- maven
  - spring
  - junit5
  - cucumber
  - failsafe
  - jacoco

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
